<Info>
  **Prerequisites**:
  - Node.js version 14 or higher
  - npm or yarn package manager
  - macOS (for testing the app)
  - Git
</Info>

Follow these steps to set up YouTube-Floater for local development.

## Getting Started

<Steps>
<Step title="Clone the repository">

Clone the YouTube-Floater repository from GitHub:

```bash
git clone https://github.com/jimbrend/YouTube-Floater.git
cd YouTube-Floater
```
</Step>

<Step title="Install dependencies">

Install all required Node.js packages:

```bash
npm install
```

This will install Electron.js and all other dependencies needed to run the app.

</Step>

<Step title="Start the development server">

Run the app in development mode:

```bash
npm start
```

The YouTube-Floater application will launch in development mode with hot-reloading enabled.

</Step>
</Steps>

## Project Structure

Understanding the codebase structure:

```
YouTube-Floater/
├── main.js           # Electron main process
├── renderer.js       # Renderer process (UI logic)
├── index.html        # Main window HTML
├── styles.css        # Application styles
├── package.json      # Dependencies and scripts
└── assets/          # Icons and images
```

## Building for Distribution

To create a distributable .dmg file for macOS:

<Steps>
<Step title="Install electron-packager">

If not already installed:

```bash
npm install electron-packager --save-dev
```
</Step>

<Step title="Build the app">

Create a production build:

```bash
npm run build
```

This will generate a distributable application in the `dist/` directory.

</Step>
</Steps>

## Technical Implementation

### Key Technologies

**Electron.js Framework**
- `BrowserWindow`: Creates the floating window with always-on-top capability
- `IPC`: Handles communication between main and renderer processes
- `webPreferences`: Configures security and node integration

**YouTube Integration**
- YouTube Embed Player API for video playback
- URL parsing to extract video IDs
- Embed parameters for custom controls

**Window Management**
- Custom drag handlers for repositioning
- Corner-snapping algorithm for quick positioning
- Opacity control using CSS and Electron APIs

## Development Tips

<AccordionGroup>
  <Accordion title="Debugging">
    Open Developer Tools while running the app:
    - macOS: `Cmd + Option + I`
    - Or add to main.js: `mainWindow.webContents.openDevTools()`
  </Accordion>

  <Accordion title="Testing changes">
    The app supports hot-reloading in development mode. Simply save your changes and the app will automatically reload.
  </Accordion>

  <Accordion title="Working with YouTube API">
    Be aware of YouTube's CORS restrictions. The app uses the official YouTube Embed Player API which has some limitations on rapid reloads.
  </Accordion>
</AccordionGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="App won't start">
    1. Ensure you're running Node.js v14 or higher: `node --version`
    2. Delete `node_modules/` and reinstall: `rm -rf node_modules && npm install`
    3. Clear npm cache: `npm cache clean --force`
  </Accordion>

  <Accordion title="YouTube videos not loading">
    This may be due to YouTube's CORS restrictions. Try:
    - Using a different video URL
    - Clearing the app cache
    - Checking your internet connection
  </Accordion>

  <Accordion title="Build fails on macOS">
    Ensure you have the latest Xcode Command Line Tools:
    ```bash
    xcode-select --install
    ```
  </Accordion>
</AccordionGroup>
